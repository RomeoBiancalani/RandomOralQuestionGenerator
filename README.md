# RandomOralQuestionGenerator

A program designed by some students for teachers to handle oral tests.

## To Run

```
git clone https://github.com/RomeoBiancalani/RandomOralQuestionGenerator
cd randomoralquestiongenerator
npm install
npm run dev
```
